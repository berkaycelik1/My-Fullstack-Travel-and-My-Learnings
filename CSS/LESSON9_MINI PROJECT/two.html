CSS Modern Layout Rehberi

1. Bölüm: CSS Box Model (Kutu Modeli)

    Başlık:

    Her şey bir kutudur!

    İçerik:

    CSS’te her element aslında görünmez bir kutunun içindedir. Bu kutuyu kontrol etmek, tasarımın temelidir.

  • Content: Yazı veya resmin olduğu merkez.
  • Padding: İçerik ile kenarlık arasındaki iç boşluk.
  • Border: Kutunun çerçevesi.
  • Margin: Kutunun diğer kutularla arasındaki dış boşluk.

2. Bölüm: Flexbox ile Esnek Tasarımlar

Başlık: Tek Boyutlu Mucize: Flexbox

İçerik: Elemanları yan yana veya alt alta dizmek artık çok kolay. display: flex; komutuyla bir kapsayıcı oluşturduğunuzda, içindeki çocuk elemanları istediğiniz gibi hizalayabilirsiniz.

  • justify-content: Elemanları yatayda (sağa, sola, merkeze) dağıtır.
  • align-items: Elemanları dikeyde hizalar.

3. Bölüm: Responsive (Duyarlı) Tasarım

Başlık: Web Siteniz Her Ekrana Uygun mu?

İçerik: Modern web siteleri hem dev monitörlerde hem de küçük telefon ekranlarında kusursuz görünmelidir. Bunu Media Queries kullanarak yaparız. Örneğin, ekran genişliği 600px’in altına düştüğünde yan yana duran kutuları alt alta dizebiliriz.
